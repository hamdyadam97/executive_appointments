# نظام حجز مواعيد المدير التنفيذي (Django)

نظام حجز مواعيد كامل مبني بـ Django مع قاعدة بيانات SQLite، واجهة عربية، ونظام موافقة ثلاثي المراحل.

## المتطلبات

- Python 3.10+
- Django 6.0+

## طريقة التشغيل

```bash
# 1. تثبيت Django
pip install django

# 2. التشغيل
python manage.py runserver

# 3. فتح المتصفح
http://127.0.0.1:8000
```

## بيانات تسجيل الدخول

| الدور | اسم المستخدم | كلمة المرور |
|-------|-------------|------------|
| موظف | `employee` | `password` |
| سكرتارية | `secretary` | `password` |
| مدير تنفيذي | `executive` | `password` |

## المميزات

- 3 أدوار: موظف / سكرتارية / مدير تنفيذي
- كاليندر أسبوعي تفاعلي (8 ص - 6 م)
- نظام طلبات مع حالات: قيد المراجعة / تمت الموافقة / تم الرفض
- منع التعارض بين المواعيد
- نظام إشعارات داخلي
- لوحة تحكم سكرتارية كاملة
- جدول مدير تنفيذي (عرض فقط)
- تصميم عربي RTL بالكامل

## المسارات

| المسار | الوصف |
|--------|-------|
| `/` | التوجيه حسب الدور |
| `/login/` | تسجيل الدخول |
| `/employee/calendar/` | كاليندر الموظف |
| `/employee/requests/` | طلبات الموظف |
| `/secretary/` | لوحة السكرتارية |
| `/executive/` | جدول المدير التنفيذي |
| `/admin/` | لوحة الإدارة |

## إعادة ضبط البيانات

```bash
python manage.py setup_demo
```
